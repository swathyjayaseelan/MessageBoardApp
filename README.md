## Message Board Application (React + Node.js + TypeScript)

This repository contains the source code for a Message Board application built with:

1. Frontend: React and TypeScript
2. Backend: Node.js and TypeScript

### Getting Started

For setting up the project locally, please clone the repo and follow the below steps

### Prerequisites

Node v14.17.6 or later installed in the machine

### Back-end
1. Run npm install
2. Run npm run start
3. The server runs in port 8000

### Front-end
1. Run npm install
2. Run npm run dev
3. Open the browser and navigate to localhost:5173

### Features
1. Create a new message by providing name and message
2. View a list of all the messages
